# Dotfiles
My dotfiles, installation scripts, and packages

## The setup

The configuration/config.toml file contains all packages and dotfiles I want to display with my application.

## Installation

### Requirements

- Python3.11 >
- Python3-pip
- setuptools (via pip)
- PySide6 (via pip)
- tomlkit (via pip)

### install

```
~ git clone https://github.com/LuukHenk/dotfiles
~ cd dotfiles
~ pip install .
```

## Usage

`~ python src/run.py`
